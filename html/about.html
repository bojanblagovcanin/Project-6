Project 6 - About

Project 6 is a web development project focused on creating an online logbook using HTML, CSS, and JavaScript. This website is designed to allow users to document and track their daily activities, tasks, and goals in an easy-to-use and visually appealing interface. Through this project, we will learn essential web development skills such as creating HTML pages, designing user interfaces using CSS, and adding interactivity to the logbook using JavaScript. Additionally, the project will provide an opportunity to practice collaboration and teamwork skills through working on a group project.
